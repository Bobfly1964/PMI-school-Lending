# АРИСТОТЕЛЬ — лендинг инженерного курса с ИИ-тьютором

## Описание

«АРИСТОТЕЛЬ» — это статическая посадочная страница для образовательного проекта: инженерного курса с ИИ-тьютором для школьников. Лендинг объясняет идею интегрированного обучения физике, математике и программированию на Python, а также приглашает учеников в фокус-группу.

Проект решает задачу первичной презентации EdTech-продукта: показывает методику курса, позиционирование ИИ-тьютора, связь с подготовкой к ОГЭ/ЕГЭ и условия участия в пилотной группе.

## Основные функции

- Презентация образовательной концепции курса «АРИСТОТЕЛЬ».
- Описание интегрированной методики: физическая задача → математика → Python-решение.
- Блок о роли ИИ-тьютора в обучении и развитии самостоятельности ученика.
- Секция о привязке курса к экзаменам ОГЭ/ЕГЭ.
- Призыв к вступлению в фокус-группу с кнопкой заявки.
